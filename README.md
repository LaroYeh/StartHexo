# StartHexo
建立的第一個Hexo框下的Blog

## Build Setup
### Installing
```
# install dependencies
npm install

# change dev folder
cd blog

# get theme
git clone https://github.com/yscoder/hexo-theme-indigo.git themes/indigo

# serve start at localhost:4000
hexo server
```

## Publish
```
hexo g
```
and upload to github
